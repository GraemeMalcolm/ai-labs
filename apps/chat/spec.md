Spec for PhiChat

## Overview

PhiChat is a simple, local-first chat application that uses Microsoft's Phi-3-mini model for AI responses. It is built using JavaScript and runs completely locally in the browser. No server is required. The application is built using the following technologies:

- HTML
- CSS
- JavaScript
- Wllama (running on CPU in WASM)
- Phi-3-mini model (running on CPU in WASM)

## Features

- Chat interface with message history
- Uses Phi-3-mini model for AI responses
- Local-first application
- Simple and easy to use
